# Foundations

Before working with Microsoft Hyper-V, it's important to understand the virtualization concepts that make modern IT infrastructure and cloud computing possible.

This section provides the theoretical background needed for the practical labs in this repository. It explains how virtualization works, the role of hypervisors, the differences between Type 1 and Type 2 hypervisors, and why these technologies are fundamental to platforms like Microsoft Azure.

## Topics Covered

- What virtualization is
- Benefits of virtualization
- What a hypervisor is
- Type 1 (Bare-Metal) hypervisors
- Type 2 (Hosted) hypervisors
- Differences between Type 1 and Type 2 hypervisors
- Why virtualization is the foundation of modern cloud computing
- Why Hyper-V is an important technology for IT professionals and cloud engineers
